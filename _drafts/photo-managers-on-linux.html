In the past, I was using Picasa to upload my photos.

These are the showstoppers in other photo management applications:

Shotwell:

Shotwell is the one I like most. It is fast, stable and the user interface is very clean. It has some problems:

  • When it import pictures, it creates a date-based folder structure. It works with pre-existing pictures though and keeps the structure intact.
  • You can rename the "event" so it gets displayed as "John's wedding" instead of the date, but this information is only persisted in the database. I would not mind keeping the date structure but don't want any relevant information going into the database. A solution would be to keep the event name in the file itself (EXIF), but this is not yet implemented.

F-Spot:

  • It does not support videos
